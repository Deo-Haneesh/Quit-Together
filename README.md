**Quit-Together** is a revolutionary social wellness platform designed to help individuals overcome addiction and stress through the power of community support, gamification, and AI companionship. Unlike traditional social media that often contributes to mental health issues, Quit-Together is built with one purpose: **to heal**.

## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn
- Firebase account (for backend services)

### Installation

```bash
# Clone the repository
git clone https://github.com/Deo-Haneesh/Quit-Together.git

# Navigate to project directory
cd Quit-Together

# Install dependencies
npm install

# Start development server
npm run dev
```

### Environment Setup

Create a `.env` file in the root directory:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_HUGGINGFACE_API_KEY=your_hf_api_key
```
